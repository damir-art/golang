# Типы данных
Типы данных в Go:
- целые числа
  - int
  - int8 - 8 байт (255), сколько байт занимает в памяти
  - int16
  - int32
  - int64
  - uint* - только положительные числа или 0
- числа с плавающей точкой
  - float32
  - float64 - самый популярный, вмещающий все необходимые диапазоны чисел
- строки
- булев тип
- массивы

## Число с плавающей точкой

    var a float64  // объявление переменной
    a = 2.2        // инициализация переменной
    fmt.Println(a) // вывод значения переменной в консоль

## Строка

    var a string
    a = "Ivan"
    fmt.Println(a)

Узнаём длину строки:

    fmt.Println(len(a))
